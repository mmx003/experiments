Перейдите на Google Cloud Console.
Создайте новый проект или выберите существующий.
Перейдите в раздел "APIs & Services" -> "Credentials" (Учетные данные).
Нажмите на "Create Credentials" (Создать учетные данные) и выберите "OAuth 2.0 Client ID".
Выберите "Desktop App" в качестве типа приложения.
Скачайте файл credentials.json после создания учетных данных.
